# VORTICE — Joyeria Escultorica

Landing page para VORTICE, marca de joyeria escultorica de autor en Medellin, Colombia. Estetica Industrial Dark Luxury con paleta de cobre, laton y oxido.

**Live:** [vortice-jewelry.vercel.app](https://vortice-jewelry.vercel.app)

## Contenido

- Hero con video header
- Galeria de piezas de joyeria (fotografia de producto)
- Seccion "Sobre nosotros" con retratos de fundadores
- Animaciones scroll-reveal

## Stack

HTML + Tailwind CSS (CDN) + JavaScript vanilla. Tipografia: Bodoni Moda, Space Mono, Syne. Deploy en Vercel.
